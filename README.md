# AI Impact on IT Service Desk Operations
This project explores how AI-assisted automation can improve operational efficiency within IT Service Management (ITSM) environments.

Using simulated service desk data, a Power BI dashboard was created to analyze ticket resolution trends, SLA compliance, automation rates, and workflow optimization opportunities.

## Objectives

- Analyze IT service desk ticket patterns
- Evaluate AI-assisted ticket resolution
- Measure SLA compliance and resolution efficiency
- Explore enterprise workflow automation concepts
- Visualize operational insights using Power BI

## Key Insights

- Approximately 38% of tickets were resolved through AI-assisted workflows
- Automation reduced average resolution time for repetitive requests
- SLA compliance metrics highlighted opportunities for predictive service improvements
- Password reset and access requests represented major automation opportunities
